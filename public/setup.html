🏗 Створи Facebook-додаток
Це потрібно зробити один раз. Facebook не дає публікувати через сторонні програми без спеціального «додатку». Це безкоштовно і займає ~5 хвилин.
Покрокова інструкція:
  1. Відкрий developers.facebook.com/apps
  2. Натисни «Create App»
  3. Вибери тип: «Other» → далі «Consumer»
  4. Введи довільну назву, наприклад My Shop Bot
  5. Натисни «Create App» і підтверди пароль
  6. Зліва: Settings → Basic — тут твої коди
Відкрити Meta Developer Portal →
2
🔗 Додай адресу повернення
Щоб Facebook знав куди тебе повернути після входу — скопіюй цю адресу і встав у налаштуваннях свого додатку.
Де вставити:
  1. У своєму додатку зліва знайди «Facebook Login for Business» або «Facebook Login»
  2. Якщо немає — натисни «Add Product» і додай «Facebook Login»
  3. Відкрий Settings цього продукту
  4. У полі «Valid OAuth Redirect URIs» встав адресу нижче і натисни «Save Changes»
http://localhost:3000/auth/facebook/callback
3
🔑 Введи App ID та App Secret
Ці коди — як логін і пароль для твого Facebook-додатку. Знайдеш їх у Settings → Basic на сторінці додатку.
Тільки цифри
32 символи, Settings → Basic → App Secret

🤖 Створи Telegram-бота
Telegram-бот — це програма-помічник, від імені якої будуть надходити пости у твій канал. Створити безкоштовно, займає 2 хвилини.
Як створити бота:
  1. Відкрий Telegram і знайди @BotFather
  2. Відправ йому команду: /newbot
  3. Придумай назву бота (наприклад My Shop Bot)
  4. Придумай username для бота — він повинен закінчуватись на bot (наприклад myshop_bot)
  5. BotFather надішле тобі токен — довгий рядок з цифрами і літерами. Скопіюй його нижче.
Відкрити @BotFather →
2
🔑 Введи токен бота
Токен виглядає приблизно так: 1234567890:ABCdefGhijklmNOpqrsTUvwxyz
Отримаєш від @BotFather після /newbot

📢 Додай бота у канал і вкажи Chat ID
Бот публікує пости у твій Telegram-канал або групу. Треба додати бота як адміністратора і вказати ID каналу.
Як додати бота у канал:
  1. Відкрий свій Telegram-канал (або групу)
  2. Зайди в налаштування каналу → Адміністратори
  3. Натисни «Додати адміністратора» і знайди свого бота за username
  4. Дай боту права: «Публікація повідомлень» (мінімум)
  5. Збережи
Як дізнатись Chat ID каналу:
  • Якщо у каналу є публічний username — просто вкажи його з @, наприклад @mychannel
  • Для приватного каналу: перешли будь-яке повідомлення з нього боту @userinfobot — він покаже ID (буде від'ємне число типу -1001234567890)
Канал де з'являтимуться пости

📲 Переконайся, що Instagram прив'язаний до Facebook Сторінки
Instagram для бізнесу публікує пости через Facebook. Для цього твій Instagram-акаунт повинен бути Professional (Business або Creator) і прив'язаний до Facebook Сторінки.
Як перевірити і прив'язати:
  1. Відкрий Instagram → Профіль → Редагувати профіль
  2. Переконайся, що тип акаунту: Professional (Business або Автор)
  3. Якщо ні — Налаштування → Акаунт → Переключитися на Professional
  4. Потім: Налаштування → Акаунт → Прив'язані акаунти → Facebook
  5. Увійди у Facebook і вибери свою Сторінку
Або через Facebook:
  • Відкрий Meta Business Suite
  • Налаштування → Instagram-акаунти → Підключити акаунт
2
📘 Підключи Facebook (якщо ще не зробив)
Instagram публікує пости через Facebook API. Якщо Facebook вже підключений — Instagram підхопиться автоматично.
3
🌐 Вкажи публічну адресу сайту
Instagram завантажує фото з інтернету — йому потрібна публічна HTTPS-адреса. Якщо сайт запущено локально, потрібен тунель (ngrok) або хостинг.
Якщо запускаєш локально — використай ngrok:
  1. Встанови: ngrok.com/download
  2. Запусти: ngrok http 3000
  3. Скопіюй адресу типу https://abc123.ngrok.io
  4. Встав нижче і збережи
Якщо сайт вже на хостингу:
  • Просто вкажи його домен, наприклад https://myshop.com
Поточна адреса: не задано

🏪 Зареєструйся і відкрий магазин на Prom.ua
Prom.ua — великий маркетплейс. Для публікації товарів потрібен акаунт продавця з відкритим магазином. Реєстрація безкоштовна.
Якщо ще немає акаунту:
  1. Відкрий prom.ua/register
  2. Зареєструйся як продавець
  3. Заповни інформацію про магазин
Якщо акаунт вже є — переходь до кроку 2.
Відкрити Prom.ua →
2
🔑 Отримай API-токен
API-токен — це секретний ключ, який дає програмі право публікувати товари у твій магазин. Видається в особистому кабінеті.
Як отримати токен:
  1. Увійди в my.prom.ua
  2. Відкрий: Налаштування → API (або пряме посилання нижче)
  3. Натисни «Згенерувати токен»
  4. Скопіюй токен і встав нижче — він показується тільки один раз!
Відкрити налаштування API →
Довгий рядок символів з особистого кабінету Prom.ua

✅ Готово — товари публікуватимуться на Prom.ua!
