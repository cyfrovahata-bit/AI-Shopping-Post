📄 Facebook Сторінка
Публікувати через API можна тільки на Сторінку (не особистий профіль). Якщо вже є — пропусти.
Створити Сторінку →
2
🏗 Створи Meta-додаток (один раз)
Безкоштовно, займає ~5 хвилин. Якщо вже є — пропусти.
  1. developers.facebook.com/apps → «Create App»
  2. Назва: My Shop Bot → Next
  3. Use cases → постав галочки:
      ✓ Manage everything on your Page
      ✓ Manage messaging & content on Instagram
    → Next → Create app
Відкрити Meta Developer Portal →
3
🔑 App ID, Secret і Redirect URI
В розділі App Settings → Basic знайди App ID і App Secret. Там само додай Redirect URI.
Крок 1 — Redirect URI:
  1. Зліва → «Facebook Login for Business» → «Settings»
  2. Поле «Valid OAuth Redirect URIs» → додай адресу нижче → Save Changes
Крок 2 — App ID і App Secret:
  1. Зліва → App Settings → Basic
  2. Скопіюй App ID і App Secret (натисни «Show») у поля нижче

4
🔐 Підключи Facebook
Натисни кнопку — відкриється вікно Facebook. Підтверди доступи і вибери свою Сторінку. Все інше зробиться автоматично.
Кілька Facebook Сторінок. Вибери ту, від якої публікуватимемо:
⚠️ Сторінку не знайдено автоматично
Введи посилання на Facebook Сторінку — ID визначиться автоматично.

🤖 Отримай токен бота
Бот вже є — якщо ні, створи його за хвилину через @BotFather. Токен схожий на 123456:ABCdef...
Якщо бота ще немає:
  1. Відкрий @BotFather → команда /newbot
  2. Дай ім'я і username (закінчується на bot)
  3. Скопіюй отриманий токен нижче
❌
2
📢 Додай бота в канал як адміністратора
Щоб бот публікував пости, він має бути адміністратором каналу з правом публікації.
  1. Відкрий свій Telegram-канал → Налаштування → Адміністратори
  2. Натисни «Додати адміністратора» → введи username бота
  3. Дай право «Публікація повідомлень» → Зберегти
3
🔗 Вкажи свій канал
Публічний канал — просто @username. Приватний — перешли будь-яке повідомлення боту @userinfobot щоб дізнатись ID.

👤 Зареєструйся на Shafa.ua
Для публікації товарів потрібен акаунт продавця на Shafa.ua. Реєстрація безкоштовна.
Якщо ще немає акаунту:
  1. Відкрий shafa.ua/uk/registration
  2. Зареєструйся як продавець
  3. Підтверди email
Якщо акаунт вже є:
  • Просто введи email і пароль нижче
Відкрити Shafa.ua →
2
🔑 Введи email та пароль від Shafa.ua
Програма заходить на Shafa.ua від твого імені і заповнює форму публікації. Дані зберігаються тільки на твоєму комп'ютері.
❌
3
✅ Готово — публікуй товари!
При першій публікації програма автоматично увійде на Shafa.ua і збереже сесію. Подальші публікації будуть швидшими — повторного входу не потрібно.

Сесія зберігається у файлі shafa-session.json. Якщо вона протухла — програма увійде знову автоматично.

1
🏪 Зареєструйся і відкрий магазин на Prom.ua
Prom.ua — великий маркетплейс. Для публікації товарів потрібен акаунт продавця з відкритим магазином. Реєстрація безкоштовна.
Якщо ще немає акаунту:
  1. Відкрий prom.ua/register
  2. Зареєструйся як продавець
  3. Заповни інформацію про магазин
Якщо акаунт вже є — переходь до кроку 2.
Відкрити Prom.ua →
2
🔑 Отримай API-токен
API-токен — це секретний ключ, який дає програмі право публікувати товари у твій магазин. Видається в особистому кабінеті.
Як отримати токен:
  1. Увійди в my.prom.ua
  2. Відкрий: Налаштування → API (або пряме посилання нижче)
  3. Натисни «Згенерувати токен»
  4. Скопіюй токен і встав нижче — він показується тільки один раз!
Відкрити налаштування API →
Довгий рядок символів з особистого кабінету Prom.ua
❌
3
📂 Вибери категорію за замовчуванням (рекомендовано)
Якщо у тебе магазин одного типу товарів — задай категорію один раз. AI також намагається підібрати категорію сам на основі фото.
Не обов'язково — якщо не задати, товари потраплять у категорію яку підбере AI.
4
✅ Готово — товари публікуватимуться на Prom.ua!
Вибери «Prom.ua» у формі публікації. AI заповнить: назву, опис, ключові слова, кольори, розміри, матеріали, сезонність, стиль.

Фото завантажуються автоматично з комп'ютера. Якщо не вийде — потрібна публічна SITE_URL (вкладка Instagram).
