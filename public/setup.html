🎵 Підключи свій TikTok
Вимоги перед підключенням:
  • Звичайний TikTok акаунт підходить для підключення. Для магазину рекомендується (не обов'язково) перевести акаунт у TikTok Business Account — безкоштовно, в TikTok: Налаштування → Керування акаунтом → Перейти на бізнес-акаунт. Це дає доступ до статистики й позначення рекламного контенту.
  • Акаунт не повинен обмежувати перегляд відео (не встановлюй режим "Приватний акаунт").

Натисни кнопку — відкриється вікно авторизації TikTok. Підтвердь доступ для акаунту магазину. У тестовому режимі пости будуть видимі тільки тобі (налаштування приватності SELF_ONLY).
⚠️Адміністратор ще не налаштував TikTok App. Розгорни налаштування нижче.

🏪 Зареєструйся і відкрий магазин на Prom.ua
Prom.ua — великий маркетплейс. Для публікації товарів потрібен акаунт продавця з відкритим магазином. Реєстрація безкоштовна.
Якщо ще немає акаунту:
  1. Відкрий prom.ua/register
  2. Зареєструйся як продавець
  3. Заповни інформацію про магазин
Якщо акаунт вже є — переходь до кроку 2.
Відкрити Prom.ua →
2
🔑 Отримай API-токен
API-токен — це секретний ключ, який дає програмі право публікувати товари у твій магазин. Видається в особистому кабінеті.
Як отримати токен:
  1. Увійди в my.prom.ua
  2. Відкрий: Налаштування → API (або пряме посилання нижче)
  3. Натисни «Згенерувати токен»
  4. Скопіюй токен і встав нижче — він показується тільки один раз!
Відкрити налаштування API →
