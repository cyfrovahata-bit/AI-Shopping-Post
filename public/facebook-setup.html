Підключення Facebook & Instagram

1

Створи Facebook App

Відкрий Meta Developer Portal → «Create App» → тип «Business» або «Other» → «Consumer».

Потім: Settings → Basic → скопіюй App ID та App Secret.

2

Додай Redirect URI у Facebook App

У твоєму App: Facebook Login → Settings → Valid OAuth Redirect URIs → додай:

http://localhost:3000/auth/facebook/callback
3

Введи App ID та App Secret

Відкриється вікно авторизації Facebook. Після підтвердження повернешся сюди автоматично.
