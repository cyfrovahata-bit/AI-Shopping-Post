Data Deletion

You can request deletion of your Postly account data, products, generated posts, uploaded media references, and connected social tokens.

How To Request Deletion

Send an email to info@cyfrovahata.com.ua from the email address used in your Postly account with the subject “Data Deletion Request”.

If you are logged in, you can also use the account deletion API from the app. Deleted accounts cannot be restored.

Facebook Data Deletion Callback

Meta can call /api/data-deletion. The endpoint returns a confirmation code and this page URL.
